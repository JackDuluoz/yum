### Wargame-Event-App

# Setup

## General

1. Type `npm install -g @angular/cli`

## Client

1. Move to client folder
2. Type `npm install`
3. To open type `ng serve --open`

## Server

1. Move to api folder
2. Type `npm install`
3. Type `npm run dev`

### Notes

- Client uses SCSS
- Ensure npm v is 8.19.3
